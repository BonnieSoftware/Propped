How to report a technical issue to us

Issues can be used to keep track of bugs, enhancements, or other requests.

Any user of Propped can create an issue in our public repository.

To report an issue:

  1. Go to our issue tracker GitHub.
  2. Click New issue.
  3. If you are not signed in you will need to sign in, and if you have never used GitHub before you will need to create an account with them.
  4. Type a title and description for your issue.
  5. When you're finished, click Submit new issue.
